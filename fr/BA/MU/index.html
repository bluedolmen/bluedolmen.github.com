    Manuel de développement de BlueCourrier¶

    Introduction¶

    BlueAppStore est un catalogue de modules prêts à l'emploi permettant l'installation en 1 clic de composants par un administrateur fonctionnel.

    Ce document détaille l'ensemble des manipulations nécessaires pour l'installation d'un module disponible sur BlueAppStore.

    Attention, BlueAppStore nécessite d'avoir un accès à internet à partir de l'instance de serveur que vous configurez.

    BlueAppStore stocke différents types de modules :

    • des modules publics, accessibles à tout le monde en lecture/écriture
    • des modules ouverts, accessibles à tout le monde en lecture seulement
    • des modules privés, accessibles uniquement à son auteur

    Installation de modules publics/ouverts¶

    L'installation de modules publics/ouverts nécessite d'abord de se connecter :

    • Connectez-vous sur http://localhost:8080/share/page/marketplace
    • Identifiez-vous avec vos login et mot de passe Alfresco (admin/admin par exemple en mode test)
    • BlueAppStore s'affiche et vous présente les modules publics
    • Vous ne pouvez pas les installer

    Pour les installer, il faut vous connecter sur la marketplace :

    • Cliquez en haut à droite sur 'Your account'
    • Identifiez-vous avec vos login et mot de passe BlueAppStore
      • Votre nom s'affiche en haut à droite
      • BlueAppStore vous présente les modules publics ET privés
    • Cliquez sur un module pour voir le détail du module
    • Cliquez sur l'icône représentant une flèche vers le bas pour l'installer

    Installation de modules privés¶

    L'installation de modules privés nécessite d'abord de se connecter :

    • Connectez-vous sur http://localhost:8080/share/page/marketplace
    • Identifiez-vous avec vos login et mot de passe Alfresco (admin/admin par exemple en mode test)
    • BlueAppStore s'affiche et vous présente les modules publics
    • Cliquez en haut à droite sur 'Your account'
    • Identifiez-vous avec vos login et mot de passe BlueAppStore
      • Votre nom s'affiche en haut à droite
      • BlueAppStore vous présente les modules publics ET privés
    • Cliquez sur un module pour voir le détail du module
    • Cliquez sur l'icône représentant une flèche vers le bas pour l'installer

    Suppression de modules¶

    Après vous être connecté :

    • Cliquez sur l'icône représentant une croix pour le supprimer
    • Le module correspondant est supprimé de votre installation

    Attention, aucun contrôle de dépendances n'est effectué pour le système dans cette version de BlueAppStore. Le module que vous supprimez est peut-être utilisé par un autre module et ne pourra donc plus fonctionner.
